# challenge-digikare

## Build docker image

### Requirements

- docker
- docker-compose

### Commands

```shell
# Build app image
docker-compose build

# Start
docker-compose up --detach --remove-orphans
```

## Build 

### Requirements

- npm

### Commands

```shell
# Build app in dist folder
npm run build

# Lint
npm run lint

# Start
npm start
```
